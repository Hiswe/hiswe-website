## prerequisite

see package.json engines

- [Node.js](https://nodejs.org/en/)
- [pnpm](https://pnpm.io/)

## commands

### building dependencies

```sh
pnpm install
```

### building for production

```sh
yarn build
```

### development

```sh
yarn start
```

application will be running on http://localhost:3000
